# Contributing

Follow the official [ReactJS tutorial](https://reactjs.org/tutorial/tutorial.html) to get started.

## Installation

1. Clone the app

```sh
git clone https://github.com/AsylumAdvice/app.git
```

2. Install NPM packages

```sh
cd app
npm install
```
3. Build the project

```sh
npm start
```
